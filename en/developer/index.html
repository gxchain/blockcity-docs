# 开发者入驻

# 注册开发者账号

访问公信宝开发者中心注册登录账号。

# 进入用户中心

登录开发者中心后，点击导航右上角用户信息，选择进入用户中心。

# 账号认证

认证企业或个人信息（推荐认证企业信息，可获得更多权益），以便使用开放平台服务。

# 绑定布洛克城账号

回到用户中心，点击绑定布洛克城账号，如未安装过布洛克城Dapp，可访问布洛克城官网下载并注册账号。
